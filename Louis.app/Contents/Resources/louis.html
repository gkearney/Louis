Louis Braille Translator

This is a simple Braille translator that will generate a Braille ready file from a wide range of file formats. These include HTML, RTF, TXT, ODT, DOC and XML files. Set the options you wish to use below and press the Translate button. You will be asked for the original file and the name for your Braille ready file.
